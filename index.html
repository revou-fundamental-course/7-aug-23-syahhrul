LUAS SEGITIGA

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Dimana :

L = Luas

a = Panjang Alas

t = Tinggi

Keliling SEGITIGA

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Dimana :

K = Keliling

S1 = Sisi A-B

S2 = Sisi A-C

S3 = Sisi B-C

Hitung Keliling Segitiga
